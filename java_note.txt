* How to override equals() method?
See example below,

	@Override
	public boolean equals(Object obj) {
		// Check shallow
		if (obj == this) {
			return true;
		}

		// Check class name
		if (obj == null || obj.getClass() != this.getClass()) {
			return false;
		}

		// Check deep
		Point2D myPoint2D = (Point2D) obj;
		return this.x == myPoint2D.x && this.y == myPoint2D.y;
	}
